# Project notes

- Создан Telegram-бот на Python с `aiogram`.
- Интерфейс не использует фото.
- Не используются названия и брендинг бота-примера.
- В каталоге оставлен один товар: `Gemini link 18 month`.
- Токен BotFather добавлен в локальный `.env`.
- `ADMIN_ID` пока пустой: заявки будут подтверждаться пользователю, но не будут отправляться администратору до заполнения ID.
- Добавлен раздел `Профиль` с балансом, покупками, промокодом, транзакциями, реферальной системой, языком и возвратом назад.
- Карточка товара `Gemini Link 18 months` переписана другими словами и показывает актуальное количество ссылок из `store.json`.
- Добавлены админ-команды `/myid`, `/stock`, `/addlinks`.
- Стартовое сообщение `/start` оформлено в стиле приветственного магазина: товар, преимущества, наличие и призыв выбрать действие.
- Вместо `store.json` добавлена PostgreSQL-база через `psycopg`.
- Добавлена минимальная веб-админка `admin.py`: статистика, последние заказы и добавление ссылок пачкой.
- Добавлен `docker-compose.yml` для локального PostgreSQL.
- Драйвер базы заменен на `psycopg[binary]`, потому что он нормально устанавливается на Python 3.13 под Windows.
- Старый `store.json` удален: база теперь PostgreSQL.
- Добавлена React-админка в `admin_frontend`.
- `admin.py` теперь FastAPI JSON API для React-админки.
- Конфигурация товара хранится в таблице `product_settings` и редактируется из админки.
- Добавлены `DEPLOYMENT.md` и `Dockerfile`.
- Добавлен CI/CD: GitHub Actions, GitLab CI и `scripts/deploy.sh`.
